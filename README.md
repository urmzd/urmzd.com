# urmzd.com

Source code for **[urmzd.com](https://urmzd.com)**.

## Setup

**Prerequisites:** Node.js, npm, [just](https://github.com/casey/just)

```bash
just install
```

## Development

```bash
just dev
```

Other commands: `just build`, `just preview`, `just fmt`, `just lint`

## Structure

Blog posts are in `src/blog/` using MDX. Powered by [Astro](https://astro.build).
